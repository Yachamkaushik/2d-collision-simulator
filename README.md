# 2D Collision Simulator

A small physics simulation built with Python and Pygame.

I only developed this because I was bored one weekend and wanted to learn a little bit about programming and physics while building something enjoyable. It's not intended to be a physics engine; it's just a fun project.

## Features

- Two balls in motion
- User-defined mass and velocity
- Elastic collisions
- Inelastic collisions
- Momentum display in real time
- A basic interactive simulation

## Tech Stack

- Python
- Pygame
